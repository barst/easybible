本書簡介：

《士師記》記載了從約書亞去世到以色列建立王國之間約三百多年的混亂時期。本書的核心主題是「以色列的惡性循環」與「神的拯救」。書中反覆出現一個循環模式：「以色列人行耶和華眼中看為惡的事 → 神將他們交在仇敵手中 → 他們受苦、呼求神 → 神興起一位士師來拯救他們 → 國中太平 → 士師死後，百姓又去行惡」。士師記描繪了一幅黑暗的圖畫，顯明了當「各人任意而行」時，社會和信仰的崩潰。然而，即使在人的悖逆中，神仍然施行祂的憐憫和拯救。

祝福禱告：

願那在人的軟弱和悖逆中仍施行拯救的神，保守您的心懷意念。願您能從士師記的歷史中學會功課，不任意而行，而是讓基督在您的生命中作王，活出順服與聖潔的生命。阿們！
